# Sistema de Gestión Integral - Panadería "El Oaxaqueño"

Este proyecto es una aplicación web y móvil (Full-Stack) desarrollada para gestionar el flujo completo de una panadería, desde el punto de venta, pasando por la producción en cocina (Jefe Obrador), hasta el control de inventario y la logística de entrega con captura de firma digital.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** Vue.js 3 (Composition API) + Vite
* **Backend / Base de Datos:** Firebase (Firestore Database y Authentication)
* **Mobile / Empaquetado nativo:** Capacitor (Android)
* **Extras:** HTML5 Canvas (para firmas digitales)

---

## 📋 Requisitos Previos

Para poder ejecutar este proyecto localmente, el equipo debe contar con:
* [Node.js](https://nodejs.org/) (versión 16 o superior)
* npm (incluido con Node.js)
* GitHub instalado(si se desea clonar el repositorio)
